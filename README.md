Brown-Bag
=========

Installation
-----------

Install Meteor:

    curl https://install.meteor.com | /bin/sh

Usage
-----

  Setup
  
    1. Clone repo
    2. Change to directory
    3. Create settings file (https://github.com/jared-schmidt/Brown-Bag/wiki/Settings-File-Example)
    3. Run server

  To run server: 
  
    meteor --settings [path/to/settings/file.json]
